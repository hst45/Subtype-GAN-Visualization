Subtype-GAN uses the vector of the shared layer to represent complex omics data and support two clustering schemes. While the number of clusters K has been specified, GMM is used as the clustering module (the model is considered a generalization of the K-means algorithm). The features trained on BCRA is originally imported.

GMM is a soft probabilistic clustering model that allows us to describe the membership of points to a set of clusters using a mixture of Gaussian densities.This model is classically trained by an optimization procedure named the Expectation-Maximization (or EM in short).
